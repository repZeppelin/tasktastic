# TaskTastic

# Introduction
Tasktastic is an application that was built for gamified productivity for wide spread of users. The current build of tasktastic that is available to **GITHUB** is more-of a proof of concept for the application.
Tasktastic aims to provide cloud-based task-tracking which allows seamless switching between devices. The simplistic and concise UI makes it easy on the users rather overloading them with information.

# Limitations
Complex SQL processes take up very low and finite resources provided by the free database provider. The database is configured to be dumbed down and to allow the least processes to occur while maintaining usability. Should resources/a better database become available, the following features will be implemented. If no better database is available, they will not be implemented.
- Password Hashing (Requires a restructing of the database and usage of more database resources)
- Email-Account Linking (Same as above.)
- 2FA (very difficult, unsure. can be implemented should the application reach public production)

# Application Operation
Currently, the most basic systems are incorporated.
- Task System
- Market System
- Leaderboard System
- Player Inventory System

### TASK SYSTEM AND MARKET SYSTEM 
The only way to earn coins is by accomplishing tasks at different difficulty levels. 
### LEADERBOARD SYSTEM
Users can have their rank anonymously displayed. Their rankings are not publicly displayed.
### ANTI-SPEEDRUN TASK SUBMISSION SYSTEM
If a task is flagged as a suspicious task by the system, system administrators can access the approval system in place by logging into the application as well.
